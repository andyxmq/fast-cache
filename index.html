项目介绍

fast-cache-andy

短小精悍的前端暂存工具，防止内存“泄漏”

简介

特色：一次开发、多次使用，[更多...][1]

安装下载

  • 下载地址 https://github.com/andyxmq/fast-cache/releases
  • npm i fast-cache-andy
  • CDN https://unpkg.com/fast-cache-andy/release/bundle.js

快速使用

import FastCache form 'fast-cache-andy'
var cache = new FastCache();
    cache.set('a', 100);
    alert(cache.get('a'));
